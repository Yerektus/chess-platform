# ♟️ Chess Platform: Больше, чем просто игра

Этот проект представляет собой полноценный прототип шахматного стартапа, написанный на **TypeScript**. Платформа выходит за рамки обычной игры, предлагая мультиплеер в реальном времени, глубокую аналитику от ИИ, социальные функции и встроенную монетизацию. 

Уникальная ниша: **Шахматы для подготовки к логическим интервью в BigTech / Платформа для турниров на внутреннюю валюту**.

## 🚀 Ключевые возможности

*   **Мультиплеер по ссылке (WebSockets):** Мгновенное создание игровых комнат и игра с друзьями без сложной регистрации.
*   **AI Coach (ИИ-Тренер):** Детальный разбор партий после игры. ИИ анализирует ходы и подсказывает: *"Здесь ты совершил ошибку, оптимальным решением было бы пойти так..."*.
*   **Социальный слой и Лидерборды:** Глобальный рейтинг игроков с фильтрацией по городам (например, *"Топ игроков из Алматы"*).
*   **Монетизация и Бизнес-логика:** 
    *   Кнопка **"Upgrade to Pro"** для доступа к продвинутой ИИ-аналитике.
    *   Интеграция со **Stripe** для покупки уникальных кастомных скинов для шахматных фигур и досок.

## 🛠 Технологический стек

*   **Язык:** TypeScript (95%+)
*   **Сетевое взаимодействие:** WebSockets (Socket.io / ws) для реал-тайм мультиплеера
*   **ИИ Интеграция:** OpenAI API (или аналог) для AI Coach
*   **Платежи:** Stripe API

## 💻 Как запустить проект локально

### Требования
*   [Node.js](https://nodejs.org/) (версия 18+ рекомендуется)
*   Пакетный менеджер npm, yarn или pnpm

### Инструкция по установке

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/Yerektus/chess-platform.git
   cd chess-platform
   ```

2. **Установите зависимости:**
   ```bash
   npm install
   # или yarn install
   ```

3. **Настройте переменные окружения:**
   Создайте файл `.env` в корне проекта (вы можете скопировать `.env.example`, если он есть) и добавьте необходимые ключи:
   ```env
   PORT=3000
   DATABASE_URL="ваша_строка_подключения_к_БД"
   OPENAI_API_KEY="ваш_ключ_openai_для_ai_coach"
   STRIPE_SECRET_KEY="ваш_секретный_ключ_stripe"
   ```

4. **Запустите проект в режиме разработки:**
   ```bash
   npm run dev
   ```

5. **Откройте приложение:**
   Перейдите в браузере по адресу `http://localhost:3000`.
