# ♟️ Chess Platform: Больше, чем просто игра

Этот проект представляет собой полноценный прототип шахматного стартапа, написанный на **TypeScript**. Платформа выходит за рамки обычной игры, предлагая мультиплеер в реальном времени, глубокую аналитику от ИИ, социальные функции и встроенную монетизацию. 

Уникальная ниша: **Шахматы для подготовки к логическим интервью в BigTech / Платформа для турниров на внутреннюю валюту**.

## 🚀 Ключевые возможности

*   **Мультиплеер по ссылке (WebSockets):** Мгновенное создание игровых комнат и игра с друзьями без сложной регистрации.
*   **AI Coach (ИИ-Тренер):** Детальный разбор партий на основе интеграции движка Stockfish.
*   **Социальный слой и Лидерборды:** Глобальный рейтинг игроков с фильтрацией по городам.
*   **Монетизация и Бизнес-логика:** 
    *   Кнопка **"Upgrade to Pro"** для доступа к продвинутой ИИ-аналитике.
    *   Интеграция со **Stripe** для покупки уникальных кастомных скинов для шахматных фигур и досок.

## 🛠 Технологический стек

Проект организован как **монорепозиторий**, управляемый с помощью **Turborepo** и **pnpm workspaces**. Весь код написан на строгом **TypeScript** (95%+ кодовой базы).

### 🏗 Архитектура Монорепозитория
*   `apps/api` — **Backend:** REST API и WebSocket-сервер на **NestJS**. Включает аутентификацию (JWT, Passport, bcrypt), работу с базой данных через **Mongoose** (MongoDB), интеграцию платежей через **Stripe** и шахматного движка **Stockfish**.
*   `apps/web` — **Основное веб-приложение:** Построено на **Next.js 14** и **React 18**. Стилизация с помощью **Tailwind CSS**. Обмен данными в реальном времени через `socket.io-client`.
*   `apps/landing` — **Landing Page:** Отдельное Next.js приложение для лендинга с интеграцией клиентской части Stripe (`@stripe/stripe-js`).
*   `packages/*` — **Общие модули:** Разделяемые типы (`@chess-platform/types`), переиспользуемые UI компоненты (`@chess-platform/ui`) и общая шахматная логика (`@chess-platform/chess-engine`).

### ⚙️ Инструменты
*   **Менеджер пакетов:** `pnpm` (v9+)
*   **Сборка и Оркестрация:** `turbo`
*   **Тестирование:** `vitest`
*   **Линтинг:** `eslint`, `prettier`

## 💻 Как запустить проект локально

### Требования
*   [Node.js](https://nodejs.org/) (версия 20.x, как указано в engines)
*   Пакетный менеджер [pnpm](https://pnpm.io/)
*   Запущенная локальная или облачная база данных MongoDB

### Инструкция по установке

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/Yerektus/chess-platform.git
   cd chess-platform
   ```

2. **Установите зависимости:**
   Для корректной установки воркспейса используйте именно `pnpm`:
   ```bash
   pnpm install
   ```

3. **Настройте переменные окружения:**
   Вам понадобится настроить `.env` файлы (скорее всего в корне, либо в `apps/api`, `apps/web` и `apps/landing`). Примерные переменные, которые понадобятся:
   ```env
   # База данных
   MONGODB_URI="ваша_строка_подключения_mongodb"
   
   # JWT / Аутентификация
   JWT_SECRET="ваш_секретный_ключ"
   
   # Stripe
   STRIPE_SECRET_KEY="ваш_секретный_ключ_stripe"
   NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY="публичный_ключ_stripe"
   ```

4. **Запустите проект в режиме разработки:**
   Так как используется Turborepo, вы можете запустить все приложения (api, web, landing) одновременно одной командой из корня:
   ```bash
   pnpm run dev
   ```
